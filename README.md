# Chirasak Tapphae - Profile Card Portfolio 📸✨

[English Version Below](#english-version)

โปรเจกต์เว็บแนะนำตัวประวัติย่อแบบบัตรโปรไฟล์ (Profile Card) ของ **จิรศักดิ์ ทับแพ (Chirasak Tapphae)** ช่างภาพสารคดีและนักสื่อสารมวลชนรุ่นใหม่ ด้วยการออกแบบธีมมืดสุดพรีเมียม (Glassmorphism Dark Theme) ตกแต่งด้วยแสงไฟสีเขียวเนออน (Neon Green Glow)

---

## 🇹🇭 ภาษาไทย

### 🌟 ประวัติโดยย่อ (About Me)
* **ชื่อ:** จิรศักดิ์ ทับแพ (ชื่อเล่น: เดียร์ / Dear)
* **บทบาท:** ช่างภาพสารคดีและนักสื่อสารมวลชนรุ่นใหม่
* **ที่อยู่:** บางพลัด, กรุงเทพมหานคร
* **ผลงานและประสบการณ์สำคัญ:**
  * ผู้เข้าร่วมโครงการอบรม **ค่ายสารคดี ครั้งที่ 18** โดยนิตยสารสารคดี
  * **งานภาพถ่ายสารคดี:** ภาพประกอบบทความ *"อิสระหรือไม่อิสระของร้านหนังสือ (อิสระ)"* และ *"ฐานถิ่นที่สูญหาย: ประมงพื้นบ้านประแสในกระแสความเปลี่ยนแปลง"* บนเว็บไซต์ Sarakadee Magazine
  * **นิทรรศการภาพถ่าย:** ร่วมจัดแสดงผลงานในนิทรรศการ *"โลกป่วน โลกร้อน โลกรวน"* ณ หอศิลปวัฒนธรรมแห่งกรุงเทพมหานคร (BACC)
  * ชื่นชอบและสนใจในเทคโนโลยี (Technology Enthusiast)

---

### 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **HTML5:** โครงสร้างเว็บที่เป็นมิตรต่อการเข้าถึง (Semantic HTML)
* **CSS3 (Vanilla CSS):** ตกแต่งสไตล์แบบกำหนดเอง (Custom Styling)
  * ใช้เทคนิค **Glassmorphism** (แผงกระจกโปร่งแสงผ่าน `backdrop-filter: blur()`)
  * ปรับแต่งด้วยสีสันโทนมืดสลับแสงเขียวนีออนแบบพรีเมียม
* **Google Fonts:** ใช้ฟอนต์ **Outfit** มอบความรู้สึกโมเดิร์นและทันสมัย

---

### 🎨 คุณสมบัติเด่นของเว็บไซต์ (Key Features)
1. **Glassmorphism Design:** บัตรโปรไฟล์กึ่งโปร่งใสสะท้อนแสงไฟนีออนเบื้องหลังอย่างสวยงาม
2. **Neon Radial Gradient Background:** พื้นหลังโทนมืดไล่ระดับสีและเอฟเฟกต์แสงออร่าสีเขียวนีออน
3. **Responsive Web Design:** รองรับการทำงานในทุกหน้าจอ ทั้งคอมพิวเตอร์และมือถือ
4. **Clean Semantic Markup:** โค้ดสะอาดยืดหยุ่น จัดระเบียบง่าย และเข้าถึงง่าย

---

### 🚀 วิธีการเข้าชมเว็บไซต์ (How to Run)
คุณสามารถเปิดใช้งานหน้าเว็บได้ง่าย ๆ โดย:
1. ดับเบิ้ลคลิกที่ไฟล์ `index.html` เพื่อเปิดบนเว็บเบราว์เซอร์ใดก็ได้
2. หรือรันผ่าน Local Server เช่น **Live Server Extension** บน VS Code

## 🇺🇸 English Version

### 🌟 Biography
* **Name:** Chirasak Tapphae (Nickname: Dear)
* **Role:** Documentary Photographer & Media Communicator
* **Location:** Bang Phlat, Bangkok
* **Key Achievements:**
  * Alumnus of the **18th Sarakadee Documentary Camp** by Sarakadee Magazine.

  * **Exhibition:** Co-exhibited at the *"Global Warming, Global Chaos, Global Change"* exhibition at the Bangkok Art and Culture Centre (BACC).

### 🛠️ Tech Stack
* **HTML5:** Semantic structural design.
* **CSS3 (Vanilla CSS):** Fully customized layout.
  * Utilizes **Glassmorphism** styling via translucent backgrounds and `backdrop-filter` blur.
  * Premium dark theme with vibrant neon green accents.
* **Google Fonts:** Configured with the modern sans-serif **Outfit** font.
